# 📝 To-Do List App

A simple and clean To-Do List application built with React, Tailwind CSS, and Material UI.

## 🚀 Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Clean, responsive UI

## 🛠️ Built With

- **React** – Component-based UI
- **Tailwind CSS** – Utility-first styling
- **Material UI (Icons)** – Icon components

## 📦 Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/ayebmehdi2/ToDoListApp.git
   ```

2. Navigate to the project folder
   ```bash
   cd your-repo-name
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Run the development server
   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:5173`
